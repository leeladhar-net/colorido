# Images Directory

This directory contains all the images used in the CollegeFest website.

## Organization

Images are organized by event categories:
- Dance events
- Fine Arts events
- Literary events
- Singing & Music events
- Dramatics events
- Tekraft events
- Sports events

## Placeholder Images

For events that don't have specific images yet, we use placeholder images from via.placeholder.com with descriptive text.

## Sports Images

Sports event images are stored locally in this directory with descriptive filenames.
